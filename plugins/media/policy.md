# START

This bot may read channel messages but it responds only when directly tagged or replied to. It uses limited safe web lookups and keeps safety/privacy guardrails in place. Do not share secrets or sensitive personal data.

**About this bot**

- The bot can observe channel traffic to assemble 1-minute context windows.
- It only replies when directly tagged or explicitly addressed.
- It may use limited **safe web lookup** to answer tagged questions.
- It does **not** browse interactively, open files, send files, run shell commands, or take actions outside Telegram.
- It may keep limited durable memory for safe channel norms, explicit user preferences, and safe learned reply/search patterns.
- It does **not** maintain hidden personal dossiers or durable profiling of users.
- Do not share passwords, tokens, private keys, or sensitive personal data with the bot.

**Use notes**

- Tag the bot directly if you want a response.
- Some categories of requests will be refused for safety/security reasons.
